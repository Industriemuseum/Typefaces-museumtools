# Typefaces
Projecten n.a.v. de tijdelijke tentoonstelling Typefaces (Industriemuseum, 11/07/2025 - 22/02/2026) 

Project 1: 
Applicatie voor touchscreen in de tentoonstelling bestaande uit twee delen:
  
  1. Digitale ontsluiting van de houten lettercollectie. Integraal raadpleegbaar en doorzoekbaar aan de hand van filteren op type.
  2. Matching puzzle spel. Test de kennis van de bezoeker over de houten lettercollectie. Maak de juiste combinaties tussen houten letters, letterproef en type (classificatie).
 
Project 2:
Webversie van houten lettercollectie voor museumtools
